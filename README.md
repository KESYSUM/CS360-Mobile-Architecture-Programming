# CS360-Mobile-Architecture-Programming

This artifact includes my completed mobile app from Project Three. The goal of this project was to design and develop a functional mobile application while applying user-centered design and mobile development best practices. The app was built to meet specific user needs by providing a simple and easy-to-use interface that allows users to complete tasks efficiently without confusion or unnecessary complexity.

The app was designed with the user in mind by focusing on clear navigation, clean layout, and essential functionality. To support this, I created multiple screens that allow users to move through the app smoothly while performing core actions. Features such as viewing, adding, updating, and managing data were implemented to ensure the app was fully functional and practical for real use.

My approach to development was to break the project into smaller parts and build step by step. I focused on implementing core functionality first, then expanding and refining as I went. This made it easier to manage the project and fix issues early instead of dealing with larger problems later. I also used simple planning techniques, like sketching layouts beforehand, to guide development.

Testing was done continuously throughout the process to make sure each feature worked correctly. This was important because it helped identify issues early and ensured the app functioned as expected. Through testing, I was able to catch errors in logic and improve the overall reliability of the application.

During development, I had to adjust my design and approach when challenges came up, especially when adding new features or modifying the structure to better meet requirements. This required flexibility and problem-solving, which helped improve the final product.

One area I was particularly successful in was creating a clean and functional user interface that connects well with the backend logic. This demonstrates my ability to combine design and development to create a working application that meets user needs.

Overall, this project shows my ability to plan, design, develop, and test a mobile application while keeping the user experience as the main focus.
